# Quick Start

Get Scribe running in under 2 minutes.

---

## Prerequisites

- Node.js 18+
- npm 9+

---

## Install & Run

```bash
# Clone (if not already)
cd ~/projects/dev-tools/scribe

# Install dependencies
npm install

# Start development
npm run dev
```

---

## First Note

1. App opens with sidebar
2. Click **+ New Note** or press **⌘N**
3. Start typing

---

## Key Shortcuts

| Action | Shortcut |
|--------|----------|
| New note | ⌘N |
| Save | Auto (no button) |
| Focus mode | ⌘. |
| Daily note | ⌘D |
| Command palette | ⌘K |
| Close | ⌘W |

---

## Wiki Links

Type `[[` to link notes:

```
See [[My Other Note]] for details.
```

---

## Tags

Type `#` to add tags:

```
This is about #research and #methods.
```
