# Editor Guide

The HybridEditor is Scribe's core writing experience - designed for ADHD-friendly, distraction-free writing.

## Modes

### Write Mode

The default mode - a simple, clean textarea:

- No formatting buttons
- No distractions
- Just you and your words
- Markdown syntax supported

### Preview Mode

See your formatted content:

- Rendered markdown
- Clickable wiki links
- Clickable tags
- Syntax highlighting for code blocks

### Switching Modes

Press `⌘E` to toggle between write and preview mode.

## Markdown Support

Full markdown support in both modes:

### Headings

```markdown
# Heading 1
## Heading 2
### Heading 3
```

### Text Formatting

```markdown
**bold text**
*italic text*
~~strikethrough~~
`inline code`
```

### Lists

```markdown
- Bullet item
- Another item

1. Numbered item
2. Another item

- [ ] Task item
- [x] Completed task
```

### Links

```markdown
[Link text](https://example.com)
[[Wiki Link]]
```

### Code Blocks

````markdown
```javascript
function hello() {
  console.log("Hello, Scribe!");
}
```
````

### Blockquotes

```markdown
> This is a quote.
> It can span multiple lines.
```

## Wiki Links

Connect your notes:

1. Type `[[` to start a wiki link
2. Autocomplete shows matching notes
3. Select or type the note title
4. Close with `]]`

Example: `[[My Research Notes]]`

## Tags

Organize with tags:

1. Type `#` followed by the tag name
2. Autocomplete shows existing tags
3. Select or create a new tag

Example: `#research #important`

## Focus Mode

Enter ultimate distraction-free mode:

1. Press `⌘⇧F`
2. Sidebars disappear
3. Editor centers
4. Just write

Exit with `⌘⇧F` or `Escape`.

## Word Count

Always visible in the status bar:

- Current word count
- Updates in real-time
- No configuration needed

## Auto-Save

Your work is automatically saved:

- Saves on every change
- No save button needed
- No data loss
- Works even if app crashes

| Action | Shortcut |
|--------|----------|
| Toggle write/preview | `⌘E` |
| Focus mode | `⌘⇧F` |
| Bold | `⌘B` |
| Italic | `⌘I` |
| Undo | `⌘Z` |
| Redo | `⌘⇧Z` |

## Font Settings

Customize your writing experience:

1. Open Settings (`⌘,`)
2. Go to Fonts section
3. Choose from 14 ADHD-friendly fonts
4. Adjust size and line height
5. Preview in real-time

### Recommended Fonts

**Sans-Serif** (clean, modern):
- Inter
- Atkinson Hyperlegible
- Lexie Readable

**Serif** (traditional):
- Literata
- EB Garamond

**Monospace** (code-focused):
- JetBrains Mono
- Fira Code
- Berkeley Mono
