## Learning Paths

### For Academic Researchers

```
1. Project System → Research projects
2. Daily Notes → Research journal
3. Terminal → Run R/Python scripts
4. Backlinks → Literature review network
```

**Total:** ~30 minutes

### For Educators

```
1. Project System → Teaching projects
2. Mission Control → Course overview
3. Daily Notes → Lecture planning
4. Command Palette → Fast course navigation
```

**Total:** ~25 minutes

### For Developers

```
1. Browser Mode → Frontend development
2. Terminal → Git, npm, testing
3. Command Palette → Keyboard workflow
4. Project System → Multiple repos
```

**Total:** ~20 minutes
