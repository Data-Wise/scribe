## Recommendation

**Hybrid Approach (A + B):**

- **Icon mode (48px):** Use vertical icon tabs with underline (Option B)
- **Compact mode (240px):** Use horizontal pill tabs (Option A)
- **Card mode (320px):** Use large pill tabs with icons + labels (Option A)

This provides:
- Visual consistency across modes
- Maximum space efficiency in icon mode
- Clear discoverability in larger modes
- ADHD-friendly with clear active states
